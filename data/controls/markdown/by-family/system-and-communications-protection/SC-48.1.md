# SC-48.1: Dynamic Relocation of Sensors or Monitoring Capabilities

**Family:** System and Communications Protection  
**Class:** SP800-53-enhancement  
**Keywords:** Assessment, assessment plan, assurance, availability, computer security, confidentiality, control, control assessment, cybersecurity, FISMA, information security, information system, integrity, personally identifiable information, OSCAL, Open Security Controls Assessment Language, Privacy Act, privacy controls, privacy functions, privacy requirements, Risk Management Framework, security controls, security functions, security requirements, system, system security

## Control Statement
Dynamically relocate sensors and monitoring capabilities dynamically relocated are defined; to locations to where sensors and monitoring capabilities are dynamically relocated are defined; under the following conditions or circumstances: conditions or circumstances for dynamically relocating sensors and monitoring capabilities are defined;.

## Implementation Guidance
None.

## Assessment Objectives
sensors and monitoring capabilities dynamically relocated are defined; are dynamically relocated to locations to where sensors and monitoring capabilities are dynamically relocated are defined; under conditions or circumstances for dynamically relocating sensors and monitoring capabilities are defined;.

## Assessment Methods
System and communications protection policy  procedures addressing sensor and monitoring capability relocation  list of sensors/monitoring capabilities to be relocated  change control records  configuration management records  system design documentation  system configuration settings and associated documentation  system audit records  system security plan  other relevant documents or records System/network administrators  organizational personnel with information security responsibilities  organizational personnel installing, configuring, and/or maintaining the system SELECT FROM: Mechanisms supporting and/or implementing sensor relocation
